# Clearance SCIM Plugin

SCIM (System for Cross-domain Identity Management) plugin for [Clearance](https://github.com/clearance-auth/clearance) — enable enterprise user provisioning.

## Installation

```bash
npm install @clearance/scim
```

## Documentation

For full documentation, visit [clearance.com/docs/plugins/scim](https://github.com/clearance-auth/clearance).
