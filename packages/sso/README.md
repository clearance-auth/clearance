# Clearance SSO Plugin

Single Sign-On plugin for [Clearance](https://github.com/clearance-auth/clearance) — add SAML and OIDC enterprise SSO to your application.

## Installation

```bash
npm install @clearance/sso
```

## Documentation

For full documentation, visit [clearance.com/docs/plugins/sso](https://github.com/clearance-auth/clearance).
