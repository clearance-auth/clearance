# Clearance Drizzle Adapter

Drizzle ORM adapter for [Clearance](https://github.com/clearance-auth/clearance).

## Installation

```bash
npm install @clearance/drizzle-adapter
```

## Documentation

For full documentation, visit [clearance.com/docs/adapters/drizzle](https://github.com/clearance-auth/clearance).
