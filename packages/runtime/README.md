# `@clearance/runtime`

The framework-agnostic TypeScript authentication runtime used by Clearance. It provides sessions, accounts, OAuth, organization workflows, adapters, framework integrations, and a plugin surface for higher-level Clearance packages.

## Installation

```bash
npm install @clearance/runtime
```

Most applications should start with `@clearance/auth`, which configures the runtime with Clearance defaults and enterprise extensions.
